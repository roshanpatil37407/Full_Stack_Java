All Types of HTML Lists

1. Ordered List (ol)

Default numbered list:

  1. Boil water
  2. Add tea leaves
  3. Add sugar and milk
  4. Serve hot

Different numbering styles:

  1. Step One
  2. Step Two
  3. Step Three
  1. Option one
  2. Option two
  1. Chapter One
  2. Chapter Two
  1. Part one
  2. Part two

2. Unordered List (ul)

Default bullet list:

  • Milk
  • Bread
  • Butter
  • Eggs

Different bullet styles:

  • Square bullet
  • Example item
  • Circle bullet
  • Example item
  • No bullet
  • Example item

3. Definition List (dl)

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
JavaScript
Programming language for web interactivity

4. Nested Lists

Lists inside other lists:

  1. Morning
    • Wake up
    • Brush teeth
    • Exercise
  2. Work
    • Meetings
    • Projects
  3. Evening
    1. Dinner
    2. Relax
    3. Sleep
